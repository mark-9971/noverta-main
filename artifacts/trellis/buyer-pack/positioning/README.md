# Noverta Buyer Positioning Pack

A grounded positioning + demo package for Noverta. Every claim in here is
backed by code that ships in `artifacts/trellis` today, not the roadmap.

## Files

- **`00-EXECUTIVE-SUMMARY.md`** — one page. The 60-second read.
- **`01-POSITIONING.md`** — one-sentence pitch, 30-second pitch (verbatim),
  2-minute demo narrative (verbatim), category framing, taglines.
- **`02-DEMO-NARRATIVE.md`** — click-by-click, 10 screens, real routes.
- **`03-COMPETITIVE-DIFFERENTIATION.md`** — Frontline, PowerSchool, Embrace,
  IC, SameGoal, CentralReach, generic SIS — and the 5-bullet Noverta wedge.
- **`04-WHY-DISTRICTS-BUY.md`** — memo for the Director of Special Education.
- **`05-WHY-AN-ACQUIRER-BUYS.md`** — memo for corp dev / PE.
- **`06-FEATURE-MATURITY.md`** — Strong now / Pilot-ready / Roadmap, every
  item tagged with the page or module that backs it up.
- **`07-RISKS-AND-GAPS.md`** — honest list of diligence challenges and the
  founder's first-voice mitigations.

## How to use

- **Writing collateral (web copy, one-pagers, sales decks):** start in
  `01-POSITIONING.md`. The taglines, the 30-second pitch, and the
  category framing are all approved verbatim.
- **Live demos:** open `02-DEMO-NARRATIVE.md` and follow the 10-screen
  path in order. Do not improvise outside the path. The MetroWest demo
  district is the rehearsed tenant.
- **Outreach to districts:** crib from `04-WHY-DISTRICTS-BUY.md`. Lead
  with the dollar/risk story, then staff retention, then parent trust.
- **Outreach to acquirers / corp dev:** crib from
  `05-WHY-AN-ACQUIRER-BUYS.md`. Be honest about what they are not buying
  yet — the section is in the document on purpose.
- **Before any buyer or diligence call:** re-read `07-RISKS-AND-GAPS.md`
  out loud. Founder names each gap first; never let a buyer "discover"
  one.

## What this pack is not

- Not a slide deck. The existing pitch decks (`trellis-pitch`,
  `trellis-demo`, `trellis-deck`) are unchanged.
- Not new product code. No features, pages, or schemas were touched.
- Not screenshots — the 10 demo screens are referenced by route name and
  described in words. Capturing images is a follow-on task.
- Not cold-email copy or a financial model. Those are downstream of this
  source material.
