2. LANE MODEL
==================================================

- high-conflict wedge work

Background lane:
- isolated reliability work
- isolated e2e
- isolated seeder cleanup
- isolated persistence work
- buyer pack/docs
- low-conflict module work

Default coding lane count:
- 2 total most of the time
- 3 max only when the third is clearly low-conflict

Do not encourage many simultaneous coding lanes against wedge files.
