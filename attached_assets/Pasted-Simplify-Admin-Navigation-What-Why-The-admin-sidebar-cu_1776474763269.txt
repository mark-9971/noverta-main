Simplify Admin Navigation
What & Why
The admin sidebar currently exposes ~40 pages, many of which are redundant, duplicated in purpose, or not useful to admins. This task cleans up the admin nav by merging overlapping pages, promoting buried but essential pages, and removing clinical/low-value items from admin view. No routes or page files are deleted — all changes are to nav configuration and one page merge.

Done looks like
Admin sidebar has a noticeably shorter, cleaner primary nav (~25 items vs ~40)
Compliance Trends is a tab inside /compliance, not a standalone sidebar entry
Compliance Risk Report and Weekly Compliance Summary are accessible from the Reports hub and from within the Compliance page, but not as top-level sidebar entries
Compensatory Finance and Cost Avoidance are merged into a single page (tabs or combined layout)
IEP Calendar is removed from admin sidebar; its content surfaces as a tab inside IEP Meetings
District Overview (/district) and Leadership Packet (/leadership-packet) are removed from admin nav; Executive Dashboard (/executive) is kept and renamed "District Overview" or "Leadership View"
Staff Directory (/staff) is moved out of "More tools" into the primary Caseload & Service Delivery nav section
IEP Suggestions, Supervision, Resource Management, and Accommodation Lookup are removed from the admin nav entirely (routes remain, just not shown to admin role)
All removed/hidden routes still work if accessed directly — nothing is broken, just not surfaced in nav
Out of scope
Changing nav for non-admin roles (case_manager, provider, para, bcba, etc.)
Deleting any page files or routes
Redesigning any page's internal layout beyond the Compensatory Finance + Cost Avoidance merge
Tasks
Update nav-config.ts for admin role — Remove standalone entries for Compliance Trends, Compliance Risk Report, Weekly Compliance Summary, IEP Calendar, District Overview, Leadership Packet, IEP Suggestions, Supervision, Resource Management, and Accommodation Lookup from the admin nav. Promote Staff Directory into the Caseload & Service Delivery section. Rename Executive Dashboard to "Leadership View" (or "District Overview").

Add Compliance Trends as a tab in /compliance — Add a Trends tab to the existing Compliance page tab bar (the tab content already exists at /compliance/trends; wire it in as an in-page tab rather than a route navigation).

Link Risk Report and Weekly Summary from Compliance and Reports pages — Add prominent link cards or buttons on both the /compliance page and the /reports page pointing to these two report pages, so they remain discoverable without a sidebar entry.

Merge Compensatory Finance and Cost Avoidance — Combine /compensatory-finance and /cost-avoidance into a single page (tabs work well here: "Financial Tracking" and "Cost Avoidance"). Redirect /cost-avoidance to the merged page. Update any internal nav links.

Add IEP Calendar as a tab in IEP Meetings — Surface the IEP Calendar content as a "Calendar" tab within /iep-meetings. Redirect /iep-calendar to /iep-meetings with the calendar tab active.
