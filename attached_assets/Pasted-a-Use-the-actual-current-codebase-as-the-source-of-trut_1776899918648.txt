Step 5 — write report + decide cutover readiness

Must-match vs allowed-to-improve
MUST MATCH
* seed path completes successfully
* overlay wiring is real
* deterministic reruns are stable
* no-mutation / no-cheating invariants still hold
* summary artifact shape is valid
ALLOWED TO IMPROVE
* showcase density
* showcase category spread
* realism of historical data
* handling-state diversity
* summary richness
MUST EXPLAIN
Any difference in:
* total alerts
* total sessions
* comp totals
* handling distribution
* risk distribution
Classify each as:
* expected improvement
* expected modeling difference
* suspicious / needs follow-up
No unexplained differences.
